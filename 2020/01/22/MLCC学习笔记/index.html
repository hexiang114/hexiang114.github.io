    基本概念

        
    标签： 要预测的数据
        
    特征： 输入变量
        
    样本： 函数或图、表

    损失

        
    用训练结果和标签对比会产生差值（损失），将损失视为新的数据，用参数（超参数）控制他的模型使损失降到最低，对损失使用方法（此处迭代法-梯度下降法），改变损失会改变训练数据的学习方向。

    训练集、验证集、测试集

    把样本分成三分

        
    训练集： 用来训练数据 ，作用在训练模型上
        
    验证集： 用来验证训练集的结果（不参与训练否则过拟合），作用在损失模型上
        
    测试集： 仅测试最终的结果,不对模型产生影响

    特征组合

        
    以取得更强预测功能

    分箱/桶（将数据按范围划分成数等分）

        
    [[10001], [42004], [2500], [18000]]
        
    将变成以下经过分桶的特征矢量：
        
    [[1], [2], [0], [1]]

    组合特征

        
    将分箱数据组合成矩阵等模型，加强预测效果(cossed column)
    [详见 “feature_crosses”] : https://blog.csdn.net/u014061630/article/details/82937333

     

    正则化

        
    降低模型复杂度，避免过拟合

    • L2正则化
      可理解为以百分比移除权重，无法下降至 0
    • L1正则化
      可理解为以绝对值移除权重，故有可能下降致 0

     

    逻辑回归与分类

    • 逻辑回归
          
      将估计值以概率形式输出
    • 分类
      • （TP）真正例
            
        模型将正类别正确预测为正类别
      • （FP）假正例
            
        模型将负类别样本错误地预测为正类别
      • （FN）假负例
            
        模型将正类别样本错误地预测为负类别
      • （TN）真负例
            
        模型将负类别样本错误地预测为正类别
    • 阈值对准确率、精确率、召回率影响
      • 准确率
            
        （TP + TN）/（TP + TN + FP + FN）
      • 精确率
            
        TP / （TP + FP）
      • 召回率
            
        TP / （TP + FN）
         
        注！：精确率与召回率互相影响
      • 提高分类阈值。
            
        假正例数量会减少，但假负例数量会相应地增加。结果，精确率有所提高，而召回率则有所降低：
      • 降低分类阈值。
            
        假正例数量会增加，而假负例数量会减少。结果这一次，精确率有所降低，而召回率则有所提高
    • ROC
          
      绘制由（TPR）真正例率和（FPR）假正例率曲线，计算其下面积，曲线下面积对所有可能的分类阈值的效果进行综合衡量。曲线下面积的一种解读方式是看作模型将某个随机正类别样本排列在某个随机负类别样本之上的概率
      （简单理解为正例在负例右边的概率）

     

    神经网络

        
    输入(特征) -> 隐藏层 –(激活函数)-> 隐藏层 -> … 隐藏层 -> 输出(测试、训练损失)

    • 影响因素
      • 激活函数
        • S型
        • ReLU (常用)
        • 双曲正切函数
      • 正则化 ~ 正则化率
        • L2
        • L1
